  <action>
Extend `js/pages/archive.js` to fetch `data/newsletters/archived-newsletters.json` and render archived year sections into the `#archived-newsletters` container using safe DOM creation methods. Add the rendering function and call it from `initArchivePage()`.

Add a new function `renderArchivedNewsletters()` that:

1. **Finds the container**: `document.getElementById('archived-newsletters')`. If not found, return early.

2. **Fetches the JSON data**: `fetch('data/newsletters/archived-newsletters.json')`. Wrap in try/catch for network errors. On failure, show a user-friendly message in the container using `textContent`.

3. **Groups entries by year**: Iterate through `data.archivedNewsletters`, building a Map of `year -> [entries]` preserving insertion order (which is already chronological within each year).

4. **Sorts years descending**: Sort year keys from 2024 down to 2008.

5. **Renders DOM for each year using safe DOM creation methods** (document.createElement, textContent, setAttribute -- NO string-based HTML injection):

For each year, create:
- A `div` with class `archive-year-section`
- An `h2` with class `archive-year-header`, textContent set to `{year} Archive`
- A `ul` with class `archive-list`

6. **Renders each item using DOM methods**: For each entry within a year:
   - **If `status === 'available'`**: Create an `li` with class `archive-list-item`. Inside it, create an `a` element with class `archive-link`, href set to `entry.filePath`, target `_blank`. Inside the link, create an `i` with classes `fas fa-file-pdf`, then a text node with `entry.editionLabel`.
   - **If `status === 'unavailable'`**: Create an `li` with classes `archive-list-item archive-item-unavailable`. Inside it, create a `span` with class `archive-label` and textContent `entry.editionLabel`, and a `span` with class `unavailable-badge` and textContent `Not Available`.

7. **Appends to container**: After building all year sections, append each to the container element.

8. **Call from initArchivePage()**: Add `await renderArchivedNewsletters();` inside the try block of `initArchivePage()`, after the existing initializations (breadcrumb, mobile menu, service worker).

Use the existing `createLogger` for logging. Log info when rendering starts and completes with the count of years rendered.

All DOM content must use safe methods -- use `textContent` for text, `setAttribute` for attributes, `classList.add` for classes. This prevents any XSS concerns even though the data source is a trusted local JSON file.
