## Meeting Information

### Viewing the Meeting Schedule

1. Click **"Meetings"** in the navigation menu
2. The page displays:
   - Current and upcoming meeting dates
   - Meeting topics and presentations
   - Special events and auctions
   - Meeting location details

### Calendar Features

#### Adding Meetings to Your Calendar
1. Find the meeting you want to attend
2. Click the **"Add to Calendar"** button
3. Choose your calendar type:
   - Download ICS file for Outlook/Apple Calendar
   - Add to Google Calendar
   - Copy event details

#### Viewing the Full Calendar
1. Scroll to the calendar section
2. Use arrow buttons to navigate months
3. Click on a date to see meeting details
4. Today's date is highlighted

### Meeting Location

**MacArthur Park Lutheran Church**
- Building 1
- Doors open at 6:30 PM
- Most Friday nights (check calendar for exceptions)

## Newsletter Archive

### Browsing Newsletters

1. Click **"Newsletter"** in the navigation menu
2. View the list of available PHILATEX issues
3. Newsletters are organized by:
   - Year and quarter
   - Most recent first

### Reading Newsletters

1. Click on any newsletter title
2. The PDF will open in a new tab
3. Use your browser's PDF viewer to:
   - Zoom in/out
   - Search within the document
   - Download for offline reading
   - Print if desired

### Newsletter Features
- **Search**: Use the search bar to find specific topics
- **Archive**: Access historical issues dating back years
- **Download**: Save PDFs for offline reading

## Educational Resources

### Accessing Resources

1. Click **"Resources"** in the navigation menu
2. Browse available materials:
   - Beginner's guides
   - Grading standards
   - Collecting tips
   - Reference materials

### Using Resource Features

#### Filtering Resources
1. Use category buttons to filter by type:
   - **All** - Show everything
   - **Beginner** - New collector resources
   - **Advanced** - Experienced collector materials
   - **Reference** - Quick lookup guides

1. Click on any resource card
2. A modal window opens with full content

   - **Bookmark** - Save for later (uses browser storage)
   - **Print** - Print-friendly version

## Philatelic Glossary

### Using the Glossary

1. Click **"Glossary"** in the navigation menu
2. Browse terms alphabetically

### Glossary Features

#### Searching Terms
1. Type in the search box
2. Results update as you type
3. Partial matches are included

#### Understanding Entries
Each glossary entry includes:
- **Term** - The philatelic word or phrase
- **Definition** - Clear explanation

## Contact Us

### Using the Contact Form

1. Click **"Contact"** in the navigation menu
2. Fill out the required fields:
   - **Name** - Your full name
   - **Email** - Valid email address
   - **Phone** (optional) - Contact number
   - **Message** - Your question or comment

3. Click **"Send Message"**
4. Wait for confirmation message
5. We typically respond within 48 hours

### Contact Information
- **Email**: loz33@hotmail.com
- **Meeting Location**: MacArthur Park Lutheran Church
- **Meeting Time**: Fridays, doors open 6:30 PM

### Rate Limiting
To prevent spam, the contact form allows 3 submissions per hour. If you need to send additional messages, please email directly.

## Accessibility Features

### Keyboard Navigation
- **Tab** - Move between links and buttons
- **Enter** - Activate links and buttons
- **Escape** - Close modal windows
- **Arrow keys** - Navigate menus

### Screen Reader Support
- All images have descriptive alt text
- Form fields have proper labels
- Heading hierarchy for easy navigation
- Skip links to main content

### Visual Accessibility
- High contrast colors
- Resizable text
- Clear focus indicators
- No flashing content

### Getting Help
If you encounter accessibility issues:
1. Email us with details
2. Include your browser and assistive technology
3. We'll work to resolve any problems

## Tips for Best Experience

### Browser Recommendations

### Saving Content
- **Bookmarks** - Use browser bookmarks for quick access
- **PDFs** - Download newsletters for offline reading

- **Resources** - Use the bookmark feature to save favorites

### Troubleshooting

#### Page Not Loading
1. Check your internet connection
2. Try refreshing the page (Ctrl+R or Cmd+R)
3. Clear browser cache if needed

#### PDFs Not Opening
1. Ensure you have a PDF reader
2. Try downloading instead of viewing inline
3. Check browser popup blocker settings

#### Forms Not Working
1. Enable JavaScript in your browser
2. Check that required fields are filled
3. Ensure you haven't exceeded rate limits
